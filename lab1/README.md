# Lab 1 - Water Management Dashboard

This project provides a simple water management dashboard with a Next.js frontend and an optional backend API structure for water, maintenance, and history services.

## Structure
- Frontend: Next.js app router based UI
- Backend: Python service layer for API endpoints
- Data flow: frontend calls local API routes and backend services

## Getting started
1. Open the frontend folder and install dependencies.
2. Start the Next.js development server.
3. Optionally run the backend Python service.
