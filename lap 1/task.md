# Task List (งานถัดไป)

ไฟล์นี้เก็บรายการงานที่จะทำต่อสำหรับโปรเจคระบบจัดการน้ำประปาหมู่บ้าน

- เพิ่ม API CRUD สำหรับ `Meter`
- เพิ่ม authentication (JWT) และ role-based access
- เพิ่ม validation สำหรับ input (เช่น `zod`)
- เพิ่ม endpoint สำหรับการคำนวณบิล (billing)
- เขียน unit/integration tests สำหรับ `readings` และ `households`
- สร้าง frontend เบื้องต้น (ถ้าต้องการ)

ถ้าต้องการให้เริ่มงานไหน ให้ตอบชื่องานหรือหมายเลขรายการที่ต้องการให้ผมทำต่อได้เลย
