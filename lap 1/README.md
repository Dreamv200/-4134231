# ระบบวัดระดับความขุ่นของน้ำ

## ภาพรวม
ระบบนี้ถูกพัฒนาขึ้นเพื่อช่วยติดตามและตรวจสอบระดับความขุ่นของน้ำ โดยสามารถแจ้งเตือนเมื่อค่าความขุ่นสูงเกินระดับที่กำหนด และบันทึกประวัติการล้างถังและงานซ่อมบำรุง

## วัตถุประสงค์
- ตรวจวัดและแสดงระดับความขุ่นของน้ำ
- แจ้งเตือนเมื่อถึงเกณฑ์ที่กำหนดหรือพบความขุ่นสูง
- บันทึกประวัติการล้างถังและซ่อมบำรุง
- รองรับการใช้งานทั้งบนคอมพิวเตอร์และมือถือ

## คุณสมบัติหลัก
- แสดงระดับความขุ่นของน้ำแบบเรียลไทม์
- ระบบแจ้งเตือนเมื่อค่าความขุ่นสูงเกินกำหนด
- บันทึกประวัติการล้างถัง
- บันทึกประวัติการซ่อมบำรุง
- รองรับการใช้งานบนเว็บทั้ง PC และมือถือ

## เทคโนโลยีที่ใช้
- Frontend: เว็บแอปพลิเคชันสำหรับแสดงข้อมูลและแจ้งเตือน
- Backend: ระบบจัดการข้อมูลและการประมวลผล

## วิธีใช้งาน
1. เปิดเว็บระบบผ่านเบราว์เซอร์
2. ดูระดับความขุ่นของน้ำที่แสดงบนหน้าเว็บ
3. ตรวจสอบการแจ้งเตือนเมื่อมีค่าเกินเกณฑ์
4. ดูประวัติการล้างถังและซ่อมบำรุงจากระบบ

## ผู้ใช้งาน
- ผู้ดูแลระบบ
- เจ้าหน้าที่ที่รับผิดชอบการตรวจสอบน้ำ

## หมายเหตุ
ระบบนี้ถูกออกแบบให้แยกส่วน Frontend และ Backend เพื่อให้มีความง่ายต่อการพัฒนาและดูแลรักษาในอนาคต
